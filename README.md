# Wordle Project

I'm historically bad at Wordle, so a professor told me I could get better at it if I understood the game's mechanics 
and coded a version of it myself. I'm still not very good, but this was fun!

## How it Works:
Players have 6 attempts to guess a 5-letter word. After each guess, the program provides feedback indicating correct letters and positions.

## Files
- `wordle.py` – main game program
- `words.txt` – list of possible solution words

## How to Run
1. Clone the repository
2. Run:

python wordle.py
